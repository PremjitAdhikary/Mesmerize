Bubble Sort
This is a O(n2) algorithm. The 'step' version draws the array to show the changes for the inner loop. The 'loop' version draws the array every after the inner loop calculations are done.
Essentially, 'step' version draws at O(n2) and 'loop' version draws at O(n).

Selection Sort
Another O(n2) algorithm. Just like Bubble Sort, we have the 'step' version and 'loop' version drawing at O(n2) and O(n) respectively.

Insertion Sort
One more O(n2) algorithm. You guessed it, easy to implement! Again, we have the 'step' version and 'loop' version drawing at O(n2) and O(n) respectively.

Quick Sort
Recursive algorithm implemented in an iterative pattern here.

Merge Sort
Another recursive algorithm implemented in an iterative pattern here.
Note that we are using a worker array here to do the calculation and then copying its contents into the original array. So the visialization will have the original array updated from worker array contents which will seem like values are disappearing from one end and appearing somewhere else.
