While the previous example we had a curve which is manipulated by a shifting phase, here we compare multiple curves with the same phase (of PI/2) but varying constants.

So what's the idea here?

We will start with a row of circles on the top and a column of circles at the side.

Each of these circles is a path of a point which runs on it at various speed of a particular ratio. For example if the first point travels at speed s, the next at 2s, then 3s and so on.

The different speed represents the constants in the equation.

Now, as the point travels on the circular path, if we draw lines (vertical for the points on the top row circular paths and horizontal for points on the leftmost column paths), they will look line oscillating lines at different speed.

Next we take the point of intersection of each pair of vertical and horizontal lines and trace the path it takes.

This traced path gives us the famed Lissajous Curve.
