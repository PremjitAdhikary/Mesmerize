Here it is when the base is moved, how does the connected segments react to it.

The game here is to use the sliders to align the tentacle such that the tip (red circle) touches the object (green disk). As soon as it does, the object will move to a new location and you start again to adjust the sliders.

PS: Increase the number of segments to up the challenge.

Here its about the motion of the connected segments of the tip to the base. The tip tries to reach out to something and how the other segments align themselves so that the front can grab or get as close as possible to the something.

Move the mouse to see the tentacle follow the mouse. Less segments will make the tentacle look robotic. More will make it fluid, almost like a rope.

The toggle anchor will either fix the tentacle to a base or let it free to roam around the screen.
