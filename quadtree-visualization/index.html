More of testing/demo segment for Quadtree.

There are 2 Quadtree types that I have out for demo here. The difference is how the items/points are stored.

  • In one, all the trees/nodes, parents and children store the items/points
  • In the other, the items/points are only stored in the leaf nodes

For insert operation, click anywhere on the viewport to add a point at that location. Or use the buttons to randomly insert points.

For query operation click anywhere to perform a range query based on the query size. Adjust the same with the slider provided.

As this is a demo, max points a Quadrant can hold is set to 5 (for pratical purposes this should be more so that the tree is not very deep in case the space is congested with points) and minimum quadrant edge is set to 20 (by default this is 5).
