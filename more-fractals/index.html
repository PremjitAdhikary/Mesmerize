A noteworthy bunch of fractals.

Cantor Set:
From Wikipedia, The Cantor set is a set of points lying on a single line segment that has a number of remarkable and deep properties. The most common modern construction is the Cantor ternary set, built by removing the middle third of a line segment and then repeating the process with the remaining shorter segments.

Cantor Set Variation with Perpendicular Lines:
The idea is to attach the smaller segments to the 2 ends at right angles.

Vicsek Snowflake:
Start with a big cross and have 5 smaller crosses on it at North, South, East, West and Center. Next have 5 on each of those smaller crosses and so on.

Koch Curve:
Start with a single line. Divide into 3 segments. Draw an equilateral triangle with the middle segment as base. Remove the base. Now repeat the step for each of the four segments. Keep repeating.

Koch Snowflake:
Start with an equilateral triangle. Draw Koch curve for each of the sides.

Koch Anti Snowflake:
Instead of sawtooth outwards, here it's inwards.

Cesaro Anti Snowflake:
Another variant of the Koch curve starting with 4 sides.

Minkowski Curve:
Minkowski Curve aka Minkowski Sausage, has the lines broken into 7 segments. The segments are perpendicular to each other.

Minkowski Snowflake:
Minkowski Snowflake aka Minkowski Island, starts with 4 sides.
