Consider this as an extension/spin-off of another of my project KombatSim.

In that project a number of different fighters are made to fight with each other. The idea being to touch upon all design patterns. The simulated fights were logged out into json files and the same is read here to animate them.

Consider this as My Coding Universe (MCU) where I aim to have cross overs among my projects.

Now to explain the controls...

  • Fight: Select type of fight.
    • Fair: Where both fighters have equal opportunity to win.
    • Biased: Where one of the fighter has huge advantage.
  • Fighter Class: Select class of fighter.
    • Amateur: The most basic of fighters.
    • Pro: Much more stronger and efficient than the amateurs.
    • Enhanced: Pro fighters enhanced with armor/weapon etc.
    • Super: These are fighter with godly powers.
  • Fighter: Select a fighter from the class.
  • Vs: Select a vs fighter from the class.
  • Select Fight: Select among available fights.
  • Load the selected fight.
  • Play!!! You can also use the same button to pause/unpause a fight.
